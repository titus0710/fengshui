# CHANGELOG

## [0.1.0] — 2026-05-09

### 功能
- 户型图上传与 AI 识别（Qwen3-VL），自动识别房间、朝向、格局
- 可编辑户型图：支持选择/矩形/异型绘制，添加房间，重命名
- 玄空飞星排盘：二十四山朝向 → 九宫飞星图，含替卦标注
- 地理环境分析：高德地图接入，获取周边水系、道路、建筑、特殊环境
- 综合风水评估：AI（DeepSeek）结合户型、飞星、环境三维度生成详细解读
- AI 问答：基于当前评估结果自由提问，上下文关联
- PDF 报告导出：含户型图、九宫飞星格盘、方位罗盘、详细风水解读
- 浮动打赏入口：右下角可关闭浮动框，5 分钟冷却后自动重新显示

### 修复
- 修复线上 `/api/analyze` 超时返回 HTML 导致前端 JSON 解析失败
- 修复线上部署后前端页面缓存导致新功能不显示
- 修复 PDF 中文乱码（恢复 Google Fonts 加载 + 服务器安装中文字体）
- 修复线上 Nginx 代理缓存首页导致更新不可见
- 修复 DeepSeek/高德返回非 JSON 响应时前端报错不明确
- 修复添加房间、重命名功能在部署后不可用的缓存问题

### 基础设施
- PM2 进程管理 + 宝塔面板
- Nginx 反向代理（api/analyze 路径 300s 超时、首页禁用缓存）
- 腾讯云轻量服务器部署文档
